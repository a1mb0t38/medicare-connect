## 📖 Project Overview

**MediCare Connect** is a full-stack healthcare management system that simplifies the process of connecting patients with doctors through an online appointment platform.

Patients can search doctors, book appointments, make secure online payments, manage their appointments, and leave reviews. Doctors can manage schedules, appointments, and prescriptions, while administrators oversee users, doctors, payments, and analytics.

The project emphasizes security, scalability, responsive UI, and role-based access control.

# ⚙️ Installation

## Clone the repositories

```bash
git clone https://github.com/a1mb0t38/medicare-client.git

git clone https://github.com/a1mb0t38/medicare-server.git
```

---

## Client Setup

```bash
cd medicare-client

npm install

npm run dev
```

---

## Server Setup

```bash
cd medicare-server

npm install

npm run dev
```

---

# 🔑 Environment Variables

## Client (.env.local)

```env
NEXT_PUBLIC_API_URL=

NEXT_PUBLIC_FIREBASE_API_KEY=

NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=

NEXT_PUBLIC_FIREBASE_PROJECT_ID=

NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=

NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=

NEXT_PUBLIC_FIREBASE_APP_ID=

NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
```

---

## Server (.env)

```env
PORT=

MONGODB_URI=

DB_NAME=

JWT_SECRET=

BETTER_AUTH_SECRET=

BETTER_AUTH_URL=

STRIPE_SECRET_KEY=
```
